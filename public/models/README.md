# Model Slots

Place exported GLB assets here:

- `crt-computer.glb` for the retro CRT computer.
- `cd-player.glb` for the CD player.
- `books.glb` for the paper/book object.
- `decor.glb` for optional small decor.

The scene reads paths and transforms from `src/data/models.ts`. If a file is missing,
the site renders a fallback geometry so the interactions still work.
